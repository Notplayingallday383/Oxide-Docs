# <span style="color: #32ae62;">Liquor Compatability</span>

Liquor is our Compatability layer for Anura. We named it liquor because its similar to how wine works.

## <span style="color: #32ae62;">API Support</span>

Below we have a small chart with a list of the current supported apis in Liquor

| API | Supported | Not Supported | To be Improved/Added Later |
| :--: | :---: | :---: | :---: |
| anura.fs | Full | - | - |
| anura.registerExternalApp | Full | - | - |
| anura.notifacation | Full | - | To be Upgraded as TB Notification API Expands |
| anura.TLS | ❌ | Not Implemented | - |
| anura.x86 | ❌ | Not working correctly | - |
| anura.filePicker | ❌ | - | - |
| anura.install | Full | - | - |
| anura.python | ⚠️ | Not Officially Supported | - |

Newer apis may be implemented at a later date.
